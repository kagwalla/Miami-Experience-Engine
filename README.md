# Miami Experience Engine | 2026 Peak Portal

A high-performance, single-page "Video-First" lead capture architecture engineered for premium high-net-worth (HNW) target curation during the 2026 summer peak season in Miami. 

This asset dynamically manages traffic vectors spanning the **2026 FIFA World Cup matches** (Hard Rock Stadium), luxury hospitality packages, and automated routing into a late-summer **Miami Spice VIP** pipeline.

---

## 🛠️ Tech Stack & Architecture

This repository uses a lean, zero-maintenance static architecture designed to sit on top of automated edge infrastructure:

* **Frontend:** Semantic HTML5 hardened with native dynamic viewport optimization.
* **Styling:** Utility-first [Tailwind CSS](https://tailwindcss.com) compiled via edge CDN.
* **Icons:** [Lucide Icons](https://lucide.dev) integration for crisp, vector-based interface UI.
* **Fulfillment Pipeline:** Asynchronous native JavaScript payload delivery targeting direct ecosystem automation hooks.
